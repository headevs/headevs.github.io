Introduction to JavaScript

"JavaScript was initially created to make web pages alive”. JavaScript, the dynamic and versatile programming language, empowers you to bring your web pages to life with interactive features, seamless user experiences, and endless possibilities. Let's embark on a journey to unlock the full potential of JavaScript and elevate your coding skills.Modern JavaScript is a “safe” programming language. It does not provide low-level access to memory or the CPU, because it was initially created for browsers which do not require it.

JavaScript Statements

In JavaScript, a statement is a unit of code that performs a specific action. JavaScript programs are composed of various statements that are executed sequentially or conditionally. Here are some common types of JavaScript statements:

VARIABLE DECLARATION: let x; Declares a variable named x. Variables can be declared using let, const, or var (older version) VARIABLE ASSIGNMENT x = 5; Assigns the value 5 to the variable x. FUNCTION DECLARATION: function myFunction() { // code to be executed }Declares a function named myFunction. CONDITIONAL STATEMENTS (IF-ELSE): if (condition) { // code to be executed if the condition is true } else { // code to be executed if the condition is false }Executes different blocks of code based on a specified condition. LOOPS (FOR, WHILE): for loop: for (let i = 0; i < 5; i++) { // code to be executed in each iteration } WHILE LOOP: while (condition) { // code to be executed while the condition is true }Iterates through a block of code repeatedly. SWITCH STATEMENT: switch (expression) { case value1: // code to be executed if expression equals value1 break; case value2: // code to be executed if expression equals value2 break; default: // code to be executed if expression doesn't match any case }Executes different code blocks based on the value of an expression. TRY-CATCH STATEMENT (ERROR HANDLING): try { // code that might throw an error } catch (error) { // code to handle the error }Wraps code that might throw an error in a try block and provides a mechanism to handle the error in the catch block. BREAK AND CONTINUE: for (let i = 0; i < 10; i++) { if (i === 5) { break; // exits the loop when i equals 5 } // code to be executed in each iteration }break is used to exit a loop, and continue is used to skip the rest of the loop code for the current iteration.

Single Line Comments

Single line comments start with //. Any text between // and the end of the line will be ignored by JavaScript (will not be executed). This example uses a single-line comment before each code line:

// Change heading: document.getElementById("myH").innerHTML = "My First Page"; // Change paragraph: document.getElementById("myP").innerHTML = "My first paragraph.";

Multi Line Comments

Multi-line comments start with /* and end with */. Any text between /* and */ will be ignored by JavaScript. This example uses a multi-line comment (a comment block) to explain the code:

/* The code below will change the heading with id = "myH" and the paragraph with id = "myP" in my web page: */ document.getElementById("myH").innerHTML = "My First Page"; document.getElementById("myP").innerHTML = "My first paragraph.";

JavaScript Operators

JavaScript operators are symbols or keywords that perform operations on operands. Operands can be variables, values, or expressions. Here are some common types of JavaScript operators:

1. Arithmetic Operators: + (Addition) - (Subtraction) * (Multiplication) / (Division) % (Modulus, returns the remainder of a division) Example: let x = 10; let y = 3; let result = x + y; // result will be 13 2. Assignment Operators: = (Assignment) += (Addition assignment) -= (Subtraction assignment) *= (Multiplication assignment) /= (Division assignment) Example: let x = 5; x += 3; // x is now 8 (same as x = x + 3) 3. Comparison Operators: == (Equality, type coercion) === (Strict equality, checks both value and type) != (Inequality, type coercion) !== (Strict inequality, checks both value and type) > (Greater than) < (Less than) >= (Greater than or equal to) <= (Less than or equal to) Example: let a = 5; let b = "5"; console.log(a == b); // true (equality, type coerced) console.log(a === b); // false (strict equality, different types) 4. Logical Operators: && (Logical AND) || (Logical OR) ! (Logical NOT) Example: let isTrue = true; let isFalse = false; console.log(isTrue && isFalse); // false (logical AND) console.log(isTrue || isFalse); // true (logical OR) console.log(!isTrue); // false (logical NOT) 5. Unary Operators: + (Unary plus, converts an operand to a number) - (Unary minus, negates an operand) ++ (Increment) -- (Decrement) typeof (Returns a string indicating the type of an operand) Example: let num = 5; console.log(-num); // -5 (unary minus) console.log(++num); // 6 (increment) console.log(typeof num); // "number" (type of operator) 6. Ternary Operator (Conditional Operator): condition ? expr1 : expr2 (If the condition is true, returns expr1; otherwise, returns expr2) Example: let age = 20; let status = (age >= 18) ? "Adult" : "Minor";

Assignment

In JavaScript, assignment is the process of assigning a value to a variable. Variables are used to store and manage data in your programs. Here are examples of JavaScript assignment.

1. Basic Assignment (=): let x = 10; // Assigns the value 10 to the variable x 2. Addition Assignment (+=): let y = 5; y += 3; // Equivalent to y = y + 3; Result: y is now 8 3. Subtraction Assignment (-=): let z = 7; z -= 2; // Equivalent to z = z - 2; Result: z is now 5 4. Multiplication Assignment (*=): let a = 4; a *= 2; // Equivalent to a = a * 2; Result: a is now 8 5. Division Assignment (/=): let b = 10; b /= 2; // Equivalent to b = b / 2; Result: b is now 5 6. Modulus Assignment (%=): let c = 15; c %= 4; // Equivalent to c = c % 4; Result: c is now 3 7. Exponentiation Assignment (**=) (ES6+): let d = 2; d **= 3; // Equivalent to d = d ** 3; Result: d is now 8 (2^3) 8. Concatenation Assignment (+= with strings): let str = "Hello"; str += " World"; // Equivalent to str = str + " World"; Result: str is now "Hello World" 9. Assignment Destructuring (ES6+): let [x, y] = [1, 2]; // Destructuring assignment // Now, x is 1 and y is 2 let { a, b } = { a: 5, b: 10 }; // Object destructuring assignment // Now, a is 5 and b is 10

Data Types

JavaScript has several data types that are used to represent different kinds of values. Here are the main data types in JavaScript: Primitive Data Types: String: Represents text. Example: "Hello, World!". Number: Represents numeric values. Example: 42 or 3.14. Boolean: Represents true or false values. Null: Represents the absence of a value. Note that null is an assignment value that represents no value or no object. Undefined: Represents an uninitialized variable or a variable with no assigned value.

1. Undefined: let text = "Hello"; let numberValue = 42; let isTrue = true; let noValue = null; let undefinedValue; 2. Object Data Type: • Object: Represents a collection of key-value pairs, where keys are strings and values can be any data type, including other objects. Example: let person = { name: "John", age: 30, isStudent: false }; 3. Array Data Type: • Array: Represents an ordered list of values. Arrays can contain elements of any data type. Example: let colors = ["red", "green", "blue"]; 4. Function Data Type: • Function: Represents a reusable block of code that can be invoked by name. Example: function greet(name) { return "Hello, " + name + "!"; } 5. Special Data Types: • Symbol (ES6+): Represents a unique identifier. Symbols are often used as keys for object properties to avoid naming conflicts. Example: const mySymbol = Symbol("unique"); 6. Primitive Wrapper Objects: • String Object: Represents a string as an object. This is rarely used explicitly. • Number Object: Represents a number as an object. This is rarely used explicitly. • Boolean Object: Represents a boolean as an object. This is rarely used explicitly. Example: let stringObject = new String("Hello"); let numberObject = new Number(42); let booleanObject = new Boolean(true);

Function

In JavaScript, a function is a block of reusable code that performs a specific task or set of tasks. Functions are a fundamental building block in JavaScript programming, allowing you to encapsulate logic, make your code more modular, and promote code reusability. Here's the basic syntax for creating and using functions:

• functionName: The name of the function. • parameters: Input values passed to the function (optional). • code to be executed: The logic or tasks performed by the function. • return result: The value that the function returns (optional). Example: javascriptCopy code function greet(name) { return "Hello, " + name + "!"; } let greeting = greet("John"); console.log(greeting); // Output: "Hello, John!"

Object

In JavaScript, an object is a complex data type that allows you to store and organize data in key-value pairs. Objects can represent real-world entities and are a fundamental part of the language. Here's the basic syntax for creating and using objects:

Object Literal: The most common way to create an object is by using an object literal: let person = { name: "John", age: 30, isStudent: false, greet: function() { console.log("Hello!"); } }; In this example: • name, age, isStudent: These are properties of the object, each with a key and a corresponding value. • greet: This is a method (a function associated with an object) stored as a property.

String

In JavaScript, a string is a primitive data type that represents a sequence of characters. Strings are often used to store and manipulate text. Here's an overview of strings in JavaScript:

Concatenation: let firstName = "John"; let lastName = "Doe"; let fullName = firstName + " " + lastName; console.log(fullName); // Output: "John Doe"

Array

In JavaScript, an array is a data structure that allows you to store and manipulate a collection of values. Arrays can contain elements of any data type, including numbers, strings, objects, and other arrays. Here's an overview of JavaScript arrays:

Array Literal: let fruits = ["apple", "orange", "banana"]; Elements are enclosed in square brackets []. Elements can be of different data types.
